# 2026-06-24

Terminus drove two harness changes:

- `record.Entry.Sections` is the domain-extension hook. Keep `record/`
  schema-neutral; tools render their own audit material (Terminus selected
  qualities/classified findings, Mercurius nothing for parity) as markdown
  sections.
- `reviewer.ReviewRequest.WorkingDir` now explicitly supports a supervised
  code-review caller running read-only in the reviewed checkout. Pi is the safe
  default because it suppresses context files; codex/claude still need
  prompt-level precedence until adapter-level suppression exists.
